이 폴더(user_data)는 프로그램이 종료돼도 보존되는 사용자 데이터 저장소입니다.

  settings/            설정값
  logs/                실행/세션 로그
  chat/                일반챗 대화별 저장소 (예약)
  interpreter/sandbox/ [2] 샌드박스 에이전트 세션 (대화/작업 기억 JSON)
  interpreter/host/    [3] 호스트직접 에이전트 세션

인터프리터 세션 JSON 은 매 턴 자동 저장되며, 다음 실행 시 '이어하기' 로
복원할 수 있습니다. 이 폴더를 지우면 모든 작업 기억이 사라집니다.
